Budget Tracker (Flask)

A simple web app to manage expenses and visualize budgets.

About the Project

This Flask-based Budget Tracker helps users log income and expenses, store them in a SQLite database, and view interactive charts for better financial planning.

Features

  • Add and manage transactions
  • SQLite database storage
  • Interactive charts for spending insights
  • Responsive and lightweight interface
